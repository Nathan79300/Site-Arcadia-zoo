Base Juridique du Traitement des Données à Caractère Personnel Le traitement des données à caractère personnel des utilisateurs repose sur les bases juridiques suivantes : Consentement : Lorsque nous obtenons le consentement des utilisateurs pour le traitement de leurs données personnelles, le fondement juridique est l'article 6, paragraphe 1, point a du RGPD. Exécution d'un Contrat : Si le traitement des données est nécessaire à l'exécution d'un contrat auquel l'utilisateur est partie prenante, ou pour la réalisation de mesures précontractuelles, le fondement juridique est l'article 6, paragraphe 1, point b du RGPD. Obligations Légales : Lorsque le traitement est nécessaire pour remplir une obligation légale, le fondement juridique est l'article 6, paragraphe 1, point c du RGPD. Intérêts Vitaux : Si le traitement est nécessaire pour protéger des intérêts vitaux de l'utilisateur ou d'une autre personne physique, le fondement juridique est l'article 6, paragraphe 1, point d du RGPD. Intérêt Légitime : Si le traitement est nécessaire aux fins d’intérêts légitimes poursuivis par notre société ou par un tiers, et que ces intérêts ne sont pas supplantés par les droits et libertés fondamentaux de l'utilisateur, le fondement juridique est l'article 6, paragraphe 1, point f du RGPD. 3. Collecte des Données Générales Les données à caractère personnel des utilisateurs sont collectées dans le cadre de l'utilisation de nos services. Ces données sont conservées uniquement pendant la durée nécessaire à l’accomplissement des finalités pour lesquelles elles ont été collectées. Au-delà de cette période, les données seront supprimées, anonymisées, ou leur accès sera restreint, sauf si une conservation plus longue est exigée par la loi. 4. Droits des Utilisateurs Les utilisateurs disposent des droits suivants en matière de données personnelles : Droit d’accès : Vous avez le droit de demander l'accès aux données personnelles que nous détenons à votre sujet. Droit de rectification : Vous pouvez demander la correction de toute donnée personnelle inexacte ou incomplète. Droit à l'effacement : Vous pouvez demander la suppression de vos données personnelles dans certains cas, par exemple si les données ne sont plus nécessaires aux fins pour lesquelles elles ont été collectées. Droit à la limitation du traitement : Vous avez le droit de demander la limitation du traitement de vos données personnelles sous certaines conditions. Droit à la portabilité des données : Vous pouvez demander que vos données personnelles vous soient fournies dans un format structuré, couramment utilisé et lisible par machine, ou qu’elles soient transférées à un autre responsable du traitement. Droit d’opposition : Vous avez le droit de vous opposer au traitement de vos données personnelles, notamment en cas de prospection commerciale. Pour exercer ces droits, veuillez nous contacter à l’adresse suivante : arcadia.bretagne@gmail.com. Nous traiterons votre demande dans les plus brefs délais, en respectant les délais légaux. 5. Sécurité des Données Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger les données personnelles contre la destruction, la perte, l'altération, la divulgation non autorisée ou l'accès non autorisé. Cependant, il est important de noter qu'aucun système de transmission ou de stockage de données n'est totalement sécurisé, et nous ne pouvons garantir la sécurité absolue de vos données. 6.
